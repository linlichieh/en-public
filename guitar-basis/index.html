Music Theory

Common chord relatives

  • C (1 3 5)
  • Cmaj7 (1 3 5 7)

  • Cm7 (1 b3 5 b7)
  • Csus2 (1 2 5)
  • Csus4 (1 4 5)
  • C7sus (1 4 5 b7) (sus4 is also labeled simply as ‘sus’)
  • Cadd2 (1 2 3 5) = Cadd9 (1 3 5 2)
  • Cadd4 (1 2 4 5) = Cadd11 (1 3 5 4)
  • Cadd6 (1 3 5 6) = Cadd13 (1 3 5 A
  • Caug (1 3 #5)
  • Cdim (1 b3 b5)
  • C9 (1 3 5 b7 D)
  • C11 (1 3 5 b7 2 4)
  • C13 (1 3 5 b7 2 4 6)

Steps to transcribe a music

  1. Find the key
  2. Find the bass line
  3. Find the chords based on the basses

Tool - Sonic Visualiser

  • Download and install it: https://www.sonicvisualiser.org/download.html
  • Install plugins
    • https://www.vamp-plugins.org/download.html?search=key&go=Go&platform=osx
    • Queen Mary plugin set (or whatever plugin below) -> Click Vamp Plugin Pack, it contains all plugins -> Download macOS (10.12 or newer)
    • It’s a .dmg file. After installation, it will create a folder ~/Library/Audio/Plug-Ins/Vamp
    • Open Sonic Visualiser -> Click Transform, then you will see the list of plugins
